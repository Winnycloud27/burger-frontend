# Burger Builder Website - Developer Guide
This website was built using Angular CLI version 20.1.0 with SCSS support (no UI frameworks).

## Prerequisites  
Before setting up the project locally, ensure you have the following installed:
- **Node.js** (version 10+)
- **npm** (package manager)   
- **Git** (for version control)

## Installation & Local Setup

1. Clone the repository:
```bash
   git clone https://github.com/Winnycloud27/burger-frontend.git

```
2. Navigate to the project directory:

```bash
    cd burger-frontend
```

3. Start the development server:

```bash
   ng serve
```

4. Open the application in your browser:
   
   Go to http://localhost:4200 to view the site.
