# BurgerFrontend

Welcome to the **BurgerBuilder** project! This is an Angular-based web application that allows users to **customize their burgers** and also choose from a **predefined set of burgers**. With a user-friendly interface, the app ensures that users can explore and create delicious burger combinations with some ingredient limitations to make it fun and manageable!

## Features:
- **Burger Customization**: Users can select various ingredients to customize their own burger.
- **Predefined Burgers**: A selection of pre-designed burger combinations for quick ordering.
- **Ingredient Limitations**: Each user can only choose a set number of ingredients, ensuring every burger is just the right size.
- **Responsive Design**: The app is built to be responsive and mobile-friendly for all device types.

## Development Server

To start a local development server, run the following command:

```bash
ng serve
```
Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.
